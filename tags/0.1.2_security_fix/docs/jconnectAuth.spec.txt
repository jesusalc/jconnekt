New Authentication Layer (OpenID Like..)
-----------------------------------------------------

High-end how-to
----------------------
1.from exApp send a request to Joomla with a callback URL...(include nonce) and
  the exApp name.. (stores the nonce even)
2.Then joomla do the actual login stuff....
3.If success joomla create a public key/private key and redirect the callback URL
  with the public key..
4.In the mean-time joomla stores the private-key with the timestamp and isUsed fields
5.Then in the ExApp it's call a fopen call to Joomla with the private key
  (generated using public-key) as the args
6.In there it'll check the private-keys validity and send the information

back-end how-to
-----------------------------
1.controller to do the login
2.create the table
3.private-key,public-key generation
4.create the private-key request function
5.modify elgg to do-auth
